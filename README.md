# jameslaurenti.com

Personal site for James Laurenti — PM, builder, Beverly MA.

**Live site:** [jameslaurenti.com](https://jameslaurenti.com)

## Stack

- [Next.js 16](https://nextjs.org) (App Router) + TypeScript
- [Tailwind CSS v4](https://tailwindcss.com)
- Markdown content via gray-matter + remark
- Deployed on [Vercel](https://vercel.com)

## Pages

- **Home** — intro, current interests, project cards
- **About** — background, how I think about work
- **Work** — selected projects
- **Making Things** — write-ups on things I've built (in progress)
- **Contact** — Formspree-backed contact form

## Local development

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).
